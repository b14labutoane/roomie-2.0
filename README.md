# Roomie

Roomie is a friendly robot that can help people be more organized and productive.

```bash
Hello World!
```
## Features
- movement controlled by a Logitech controller
- a small screen that shows time, date and weather
- leds

## Components
- Arduino Board
- USB Host Shield
- ESP8266
- x3 Leds
- small screen
- bridge
- x2 motors
- batteries
- power bank
